# 법적 컴플라이언스

## 한국 서비스 필수 법령

| 법령 | 적용 조건 | 핵심 의무 |
|------|----------|---------|
| 개인정보보호법 (PIPA) | 이메일 등 개인정보 수집 시 | 처리방침 공개, 수집 동의, 파기 절차 |
| 정보통신망법 | 온라인 서비스 | 개인정보 처리방침, 보안 조치 |
| 전자상거래법 | 유료 서비스 (결제) | 청약철회, 환불 정책, 사업자 정보 공개 |
| 저작권법 | AI 생성 결과물 | 저작권 귀속 명시 |

## 1. 개인정보 처리방침 (필수)

`/privacy` 페이지를 만들어 다음 내용을 명시:

```
1. 수집하는 개인정보 항목
   - 이메일 주소 (로그인 시)
   - 서비스 이용 기록 (생성 이력, 결제 내역)
   - IP 주소, 접속 로그

2. 수집 및 이용 목적
   - 회원 식별 및 서비스 제공
   - 결제 처리
   - 서비스 개선 및 오류 분석

3. 보유 및 이용 기간
   - 회원 탈퇴 시까지
   - 전자상거래법에 따른 거래기록: 5년

4. 제3자 제공
   - Supabase (데이터 저장): 미국 소재
   - Stripe (결제): 미국 소재
   - Sentry (오류 추적): 미국 소재
   → 국외 이전 고지 및 동의 필요

5. 이용자 권리
   - 열람, 수정, 삭제, 처리 정지 요청 가능
   - 요청 연락처: [이메일]

6. 개인정보보호 담당자
   - 성명, 직책, 연락처 명시
```

## 2. 이용약관 (필수)

`/terms` 페이지에 다음 내용 명시:

**서비스 이용 관련:**
- 크레딧 구매 후 환불 정책 (전자상거래법: 7일 이내 청약 철회 원칙)
- 서비스 이용 제한 사유 (약관 위반, 부정 이용 등)
- 서비스 변경·중단 시 사전 고지 의무

**AI 생성 결과물 저작권 (가장 중요):**
```
제N조 (AI 생성 결과물의 권리)
① 본 서비스를 통해 생성된 HTML, 이미지 등 결과물의 저작권은 다음과 같이 처리됩니다:
   - 사용자가 입력한 제품 정보, 브랜드 요소: 사용자에게 귀속
   - AI가 생성한 카피, 이미지: 서비스 약관 및 각 AI 제공사 정책에 따름
② 생성 결과물을 상업적 목적으로 사용할 경우, 각 AI 모델 제공사의 이용약관을 준수해야 합니다.
③ 회사는 생성 결과물의 정확성, 완전성, 적법성을 보장하지 않습니다.
```

**참고: 각 AI 제공사 상업적 이용 정책**
- OpenAI (GPT-4o): 생성 결과물에 대한 소유권 사용자에게 부여 (이용약관 확인 필요)
- Anthropic (Claude): 생성 결과물 상업적 이용 허용 (이용약관 확인 필요)
- Google (Gemini): 이미지 생성 결과물 상업적 이용 정책 별도 확인 필요

## 3. 사업자 정보 공개 (전자상거래법)

결제 서비스 제공 시 하단 footer에 다음 정보 반드시 표시:

```
상호명: [회사명]
대표자: [이름]
사업자등록번호: [번호]
통신판매업 신고번호: [번호]
주소: [주소]
이메일: [이메일]
고객센터: [연락처]
```

통신판매업 신고는 관할 구청에서 신청 (온라인 사업자라면 필수).

## 4. 결제 관련 고지

```
환불 정책:
- 미사용 크레딧: 구매 후 7일 이내 전액 환불
- 사용된 크레딧: 환불 불가
- 단, 서비스 오류로 인한 크레딧 소모 시 복구 가능

크레딧 유효기간:
- 구매일로부터 1년 (또는 회원 탈퇴 시 소멸)
```

## 5. 마케팅 수신 동의

이메일 마케팅(소식지, 프로모션 안내)은 별도 동의 필요.
가입 시 체크박스로 선택적 동의 받기:

```tsx
<label>
  <input type="checkbox" name="marketing" />
  (선택) 서비스 업데이트 및 프로모션 이메일 수신에 동의합니다.
</label>
```

## 6. 만 14세 미만 이용 제한

개인정보보호법상 만 14세 미만은 법정대리인 동의 필요.
가입 시 "만 14세 이상입니다" 확인 체크박스 추가.

## 7. 회원 탈퇴 및 데이터 삭제

개인정보보호법: 탈퇴 즉시 또는 일정 기간 내 개인정보 파기 의무.

```ts
// app/api/account/delete/route.ts
export async function DELETE() {
  const user = await requireAuth()

  // 1. 사용자 프로젝트 데이터 삭제 (Storage + DB)
  const projects = await listProjects(user.id)
  for (const p of projects) {
    await deleteProject(p.id)
  }

  // 2. 결제 정보 삭제 (Stripe Customer 삭제)
  await stripe.customers.del(user.stripeCustomerId)

  // 3. Supabase Auth 계정 삭제
  await supabase.auth.admin.deleteUser(user.id)

  return NextResponse.json({ deleted: true })
}
```
